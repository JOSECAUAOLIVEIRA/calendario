# calendario
O Calendário de Tarefas é um sistema completo desenvolvido em Python usando Programação Orientada a Objetos para gerenciar suas atividades diárias e semanais. O sistema permite organizar, editar, excluir e acompanhar tarefas com um sistema inteligente de avisos para atividades pendentes.

FUNCIONALIDADES PRINCIPAIS
Adicionar tarefas com data e hora específicas

Editar qualquer informação da tarefa (descrição, data, hora)

Excluir tarefas não mais necessárias

Marcar como concluída para acompanhar progresso

VISUALIZAÇÃO ORGANIZADA
Visualização por dia: Veja todas as tarefas de um dia específico

Visualização semanal: Acompanhe toda a semana de uma vez

Ordenação automática por data e hora

Símbolos visuais (✓ ✗) para status de conclusão

SISTEMA DE AVISOS INTELIGENTE
Avisos automáticos ao iniciar o programa

Notificações de tarefas pendentes do dia atual

Contador de tarefas não concluídas

Lembretes para não esquecer compromissos

PERSISTÊNCIA DE DADOS
Salvamento automático em arquivo JSON

Dados mantidos entre execuções do programa

Backup automático das informações

Recuperação de dados em caso de reinício
